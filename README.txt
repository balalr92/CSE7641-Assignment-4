1. Download and Install Java SDK (https://www.oracle.com/technetwork/java/javase/downloads/jdk10-downloads-4416644.html)
2. Download Jython 2.7.0 (http://search.maven.org/remotecontent?filepath=org/python/jython-installer/2.7.0/jython-installer-2.7.0.jar)
3. Add Jython and Java to your PATH environment variable

4. The Python files are located https://github.com/balalr92/CSE7641-Assignment-4
5. One other thing we need to run this is the BURLAP.jar file. That can be downloaded from https://github.com/JonathanTay/CS-7641-assignment-4/tree/master/Solution/burlap.jar
6. Place the files from both locations in the same folder

7. Open Command Prompt
8. Navigate to the folder where the .py and .jar files are located
9. Execute the command jython easyGW.py (For small Grid)
10. Execute the command jython hardGWGW.py (For Large Grid)
11. Execute the command jython varsizeGW.py (To see the effect of Size on Running time and iterations)
